# Support

## Bug Reports

Please open an issue on [GitHub Issues](https://github.com/rudra496/termmind/issues) with:

- Python version (`python --version`)
- TermMind version (`termmind --version`)
- Steps to reproduce
- Expected vs actual behavior

## Feature Requests

Open an issue with the `enhancement` label describing the feature and use case.

## Questions

For usage questions, open a GitHub Discussion if available, or post an issue with the `question` label.
