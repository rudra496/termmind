# Roadmap

## Planned

- **Multi-turn file editing** — Apply multiple edits in a single AI turn
- **Image support** — Send screenshots/images to vision-capable models
- **MCP integration** — Model Context Protocol for tool use
- **Embedded database** — Persistent vector store for code context across sessions
- **Web UI** — Optional browser-based interface for chat sessions
- **Team features** — Shared configurations and snippet libraries

## Under Consideration

- Jupyter notebook cell editing
- Remote repository support (clone and analyze without local checkout)
- GitHub Actions integration (run termmind in CI pipelines)
- Additional language support for code index (C++, Ruby, PHP)
